# 梅蒂莉尔：讨伐词条说明

- 基础：CharacterMB `29`，翠属性、速度 2796、魔法普攻；采用 Lv240 / EX3。
- S1（CD 4）：随机 4 目标 `410% ATK` 魔法伤害，木桩折叠一次；伤害后获得 3 行动护盾 `2900130201`，数值不计（`raidIgnoredShield`）。
- S2（CD 4）：无伤害辅助；结算后为攻击优先级最高的一名友方减 2 点冷却，并施加 3 行动攻击 +85% / 防御提升同组状态 `2900260102`。防御与弱化解除不计（`raidIgnoredDefenseBuff`、`raidIgnoredDebuffCleanse`）。
- 后续：护盾、防御和净化均已有词条索引，可在承伤模型中统一启用。
