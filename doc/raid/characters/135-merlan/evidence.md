# 梅尔林：讨伐拉表接入证据

## 接入条件

- CharacterMB：`135`；流金，速度 `3018`，魔法普通攻击。
- 采用档位：Lv240、专武 EX3。S1 的 5 目标来自 EX1、440% 来自 EX3；S2 没有专武档，采用 Lv181 的最终技能等级。
- 证据：`CharacterMB[135]`、`ActiveSkillMB[135001/135002]`、`PassiveSkillMB[135003/135004]`、`data/battle-logs/y1800mrl.json`。
- 当前模型不计算防御，因此减魔防保留为可见 Boss 弱化组，倍率固定为 `0`。

## 战斗日志确认

- `y1800mrl.json` 的第 1 回合 `TurnStartPassiveResults` 中，`AttackUnitGuid`、`TargetUnitGuid` 与 `GranterGuid` 均为梅尔林的 `31`：妖精 `13500300101` 为 `SkillCategory: 4`，护盾 `13500340201` 与守护 `13500430101` 均为 `SkillCategory: 2`。
