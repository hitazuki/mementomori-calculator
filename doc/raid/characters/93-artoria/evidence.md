# 阿尔托莉雅：讨伐词条说明

- 基础：CharacterMB `93`，绯属性、速度 3572、物理普攻；采用 Lv240 / EX3。
- 自伤联动：任意上阵角色触发 `selfDamage` 时，“正义” +1，最多 4 层；每层自身攻击 +10%。本人的 S1/S2 自伤发生在伤害前，故当次也会获得一层。
- 开战：速度最低的其他友方获得 2 行动 `0.5 × ATK_阿尔托莉雅 / ATK_目标` 符号攻击项（`9300430101`）；若其为绯属性，另获速度 +20%（`9300440201`）。
- S1（CD 4）：自伤后 `6 × 520% ATK`；目标为翠属性时，每段按70%概率施加1回合气绝 `9300120202`。索尼娅为蓝属性不触发；光士为翠属性，按概率开关的确定场景处理。气绝参与弱化联动，但不让木桩跳过行动。
- S2（CD 4）：自伤后 `7 × (540% + 150% × 正义层数)`，最高 1140%。
- `r1820` 确认正义为 `EffectGroup 9300320101 / SkillCategory=4`，任意队友自伤都会在该次事件内立刻加层；自身S1/S2自伤同样在主体伤害前加层。开战多重屏障为 `9300330201 / SkillCategory=4`。
- 同一日志中，圣剑之鞘选择实际速度最低的アイシェ（Guid 1），攻击与速度状态均持续2次目标行动，施加者为阿尔托莉亚（Guid 31）。
- `r1820` 的敌方阵容没有满足阿尔托莉亚气绝条件的翠属性目标，因此没有玩家组成功样本。`g1820` / `g1820-2` 的敌方镜像组 `550409300120202` 有多次成功附带记录：`EffectGroupType=1 / SkillCategory=3 / EffectType=6001 / EffectTurn=1`；该组与玩家组 `9300120202` 共享MB名称及描述键，确认其为可解除弱化。
