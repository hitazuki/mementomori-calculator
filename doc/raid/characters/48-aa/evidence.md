# A.A.：直接伤害日志证据

## 来源与身份

- 战斗日志：`data/battle-logs/41-60-d.json`。
- A.A.：`UnitId=48 / BattleCharacterGuid=41`。
- 日志面板：基础魔力 `9,260,517`，暴击伤害强化 `198%`；第1至4回合另持有 `6900140102` 暴击伤害 +20%。
- S2：`ActiveSkillId=48002`，EX3为随机4体 `980% MAG` 直接攻击。
- 强化普攻：`4800330101`，随机3体 `330% MAG` 直接攻击。

## 可精确复算的结果

### 强化普攻暴击

第4回合，A.A.的强化普攻对 `TargetUnitGuid=10` 记录 `HitType=3 / ChangeHp=112,459,718`：

```text
9,260,517 × 330% × (1.5 + 1.98 + 0.20)
= 112,459,718
```

该目标没有通用承伤增减；结果不含防御衰减。帕拉底亚的 `6300330101` 暴击监听计数同时增加，交叉确认该次直接攻击属于暴击。

### S2暴击

第6回合，S2对 `TargetUnitGuid=40` 记录 `HitType=3 / ChangeHp=315,820,671`；20%暴伤状态已到期：

```text
9,260,517 × 980% × (1.5 + 1.98)
= 315,820,671
```

该结果再次确认MAG直接攻击可暴击并使用暴击伤害强化，且不经过目标双防。

### 目标承伤降低

第3、4回合，强化普攻对 `TargetUnitGuid=20` 均记录 `HitType=1 / ChangeHp=21,391,794`。目标持有 `550405800330101`：最大HP +30%、承伤降低30%。

```text
9,260,517 × 330% × (1 - 30%)
= 21,391,794
```

因此直接攻击无视防御，但仍进入目标承伤增减管线。

## 证据边界

- 已确认：MAG直接攻击可暴击、使用暴击伤害强化、受目标承伤降低影响、无视防御/物防/魔防。
- 未隔离确认：攻击者输出增伤、目标承伤增加，以及两者同时存在时是否按当前项目的统一 `damageRate` 加算。
- 光士的STR伤害同样由技能声明为直接攻击，当前复用通用管线；A.A.日志不能排除STR直接攻击存在额外特例，仍需光士自身日志确认。
