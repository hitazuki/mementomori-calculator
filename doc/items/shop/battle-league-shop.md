# 古竞技商城

> 来源：用户提供的古竞技商城截图。  
> 兑换货币：古竞技币（`ItemType=13`、`ItemId=10`、`IconId=45`）。  
> 商城母表：`TradeShopTabMB.Id=7`，游戏内页签名为「古竞技」。

## 数据口径

- 每个截图卡片按一个独立兑换档位记录；画面中相同道具、数量和价格的重复卡片也分别保留。
- `获得数量` 按道具图标右下角数量读取。
- 所有档位均显示 `限购 1/1`，因此 `limitTotal = 1`。
- `单次花费` 的单位均为古竞技币。

古竞技币价格量级较大，若按每 1 币展示会产生大量接近 0 的 CE。页面改用每 1000 古竞技币口径：

```text
ShopCE = RewardValue / 古竞技币花费 × 1000
```

因此 CE 表示“每消耗 1000 古竞技币可换回多少钻石价值”；该倍率只改变计价单位，不改变商品间的排序。
