# 封印宝箱掉落表

> 来源：用户补充的游戏内概率截图。
>
> 客户端 MB 仅能定位到 `TreasureChestMB -> TreasureChestItemMB -> LotteryRewardId`，具体奖池内容和概率不在客户端 Master 中。本表用于补足 `LotteryRewardId=38/39/40` 的截图识别结果。
>
> 评分系统按本表计算宝箱内容物期望值；同级封印钥匙按“钥匙:宝箱 = 1:2”推算，即钥匙价值为同级宝箱价值的一半。
