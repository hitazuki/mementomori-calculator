# 道具价值计算公式

## 1. 评分体系

每项道具在 `item-score-template.md` 中有一个**基准评分** `S(item)`，单位为**钻石**。其余项通过换算公式自动推导。

### 1.1 挂机资源（ItemType=10）

同类型不同时长按**时间比例**换算，只评基准项（1小时）：

| 基准项 | 其他项 | 公式 |
|--------|--------|------|
| 金币(1小时) | 2h/6h/8h/24h | `S = S_base × 小时数` |
| 经验珠(1小时) | 2h/6h/8h/24h | `S = S_base × 小时数` |
| 潜能宝珠(1小时) | 2h/6h/8h/24h | `S = S_base × 小时数` |

### 1.2 培育材料组（ItemType=10, Id=16~20）

培育材料组 = 金币 + 经验珠 + 潜能宝珠（同时间数）：

```
S(培育材料组, Nh) = S(金币, Nh) + S(经验珠, Nh) + S(潜能宝珠, Nh)
```

> 例如：培育材料组(24h) = 金币(24h) + 经验珠(24h) + 潜能宝珠(24h)

### 1.3 魔装香油（ItemType=15）

```
S(魔装高级香油) = 3 × S(魔装香油)
```

### 1.4 未鉴定符石（ItemType=17, Id=5~10）

每个等级差为 ×2：

```
S(Lv N) = 2 × S(Lv N-1)
S(Lv3) = 2 × S(Lv2)
S(Lv4) = 2 × S(Lv3) = 4 × S(Lv2)
...
```

> 只评 Lv2 即可，其余自动推导。

### 1.5 魔女的来信（ItemType=17, Id=21~25）

四种属性（蓝/红/翠/黄）**等价**，任选属性需单独评分：

| 道具 | 评分 |
|------|------|
| 魔女的来信(SR・蓝/红/翠/黄) | `S_letter` |
| 魔女的来信(SR・任选属性) | 单独评分 |

### 1.6 封印钥匙（ItemType=18）

初级/中级/上级**不可换算**，需各自评分。

### 1.7 不需评分的推导项

以下无需在评分表中出现，由公式自动计算：

- 金币/经验珠/潜能宝珠 的 2h/6h/8h/24h 变体
- 培育材料组（所有时长）
- 魔装高级香油
- 未鉴定符石 Lv3~Lv7（评 Lv2 即可）

## 2. 组合包价值

对于超值限时组合包中的任一包 `P`：

```
V(P) = Σ( S(item_i) × Qty_i )
```

即包内每种道具的评分 × 数量，求和。

## 3. 性价比 (Cost-Efficiency)

```
CE(P) = V(P) / DiamondCount(P)
```

- `DiamondCount(P)` = 包内含有的**有償ダイヤ**数量（ItemType=2, ItemId=1）
- `CE(P) > 1` 表示组合包价值超过所含钻石本身，即"划算"

## 4. 示例

假设评分：

| 道具 | 评分 | 说明 |
| --- | --- | --- |
| 钻石 | 1 | 基准 |
| 金币(1小时) | 5 | |
| 经验珠(1小时) | 10 | |
| 潜能宝珠(1小时) | 15 | |
| 魔女的心片(SR) | 200 | |
| 魔装香油 | 50 | |
| 未鉴定符石Lv2 | 100 | |

推导值：

| 道具 | 推导 |
| --- | --- |
| 金币(24h) | 5 × 24 = 120 |
| 经验珠(24h) | 10 × 24 = 240 |
| 潜能宝珠(24h) | 15 × 24 = 360 |
| 培育材料组(24h) | 120 + 240 + 360 = 720 |
| 魔装高级香油 | 50 × 3 = 150 |
| 未鉴定符石Lv7 | 100 × 2^5 = 3,200 |

计算无穷之塔 ¥11800 第 50 层包：

| 道具 | 数量 | 单价 | 价值 |
| --- | --- | --- | --- |
| 有償ダイヤ | 5,900 | 1 | 5,900 |
| 魔女的心片(SR) | 480 | 200 | 96,000 |
| 培育材料组(24h) | 20 | 720 | 14,400 |
| 魔装香油 | 9 | 50 | 450 |

```
V = 5,900 + 96,000 + 14,400 + 450 = 116,750
CE = 116,750 / 5,900 = 19.79
```

> CE ≈ 19.8，表示该礼包的实际价值约为售价的 20 倍。
