# Memento Mori 戰鬥傷害公式與數值成長解析文檔

基於 `scripts/calculator/未命名的試算表 (1).xlsx` 的逆向工程數據，本文檔整理了遊戲中各項戰鬥數值的計算公式。

> [!NOTE]
> 本文檔所列公式為逆向推導結果。數值如 `0.15`、`1.1`、`0.2` 等常數可能對應遊戲中的內置減傷、陣營克制或 PVP/PVE 專屬修正係數。

---

## 1. 基礎面板攻擊力 (表攻)

角色的最終面板攻擊力由多個部分組合乘算與加算而來。
**公式：**
```text
面板攻擊力 = (武器 + 力量轉換 + 石頭 + 魔裝) × (1 + 聖裝加成% + 套裝加成% + 專武加成%) + 等級固定屬性 + 其他固定值
```

**詳細對應：**
*   **基礎來源**：武器攻擊力、力量屬性轉換值（可能受某個係數影響）、神裝石頭、魔裝。
*   **百分比乘區**：聖裝的百分比提升、套裝效果的百分比提升、專武的百分比提升。這些百分比為**加算**。
*   **固定值**：來自等級的額外屬性和其他固定加成（通常不受百分比加成影響）。

---

## 2. 傷害核心乘區

單次技能的理論最終傷害涉及多個獨立乘區的相乘。
**實戰單擊傷害公式：**
```text
最終傷害 = 面板攻擊力 × 技能倍率 × 攻擊/增傷乘區 × 抗性/減傷乘區 × 爆擊傷害乘區 × 獨立乘區
```

**參數解析（以 R13 公式為例）：**
*   **攻擊力**：角色當前面板攻擊力。
*   **技能倍率**：例如 `5.25` (525%)。
*   **攻擊/增傷乘區**：`(1 + 攻擊增加% + 傷害增加%)`。
*   **抗性/減傷乘區**：`(1 - 0.15)`，此處的 `0.15` (15%) 推測為環境固定減傷、屬性抗性或目標的特殊減傷機制。
*   **爆擊傷害乘區**：`(1.5 + 爆傷增加% + 其他爆傷增益)`，其中 `1.5` 為遊戲的基礎爆擊傷害 (150%)。
*   **獨立乘區**：例如公式中的 `1.1`，推測為**陣營克制** (10% 獨立增傷) 等最終傷害放大係數。

---

## 3. 防禦力與減傷率 (Damage Reduction)

遊戲採用的是非線性的防禦減傷模型，典型的 `防禦 / (防禦 + 常數)` 公式。

> [!IMPORTANT]
> 減傷率的計算依賴於一個隱藏的「防禦係數（常數）」，該係數可能與攻擊方等級或目標等級有關。

**防禦減傷率公式：**
```text
防禦減傷率 = 目標防禦力 / (目標防禦力 + 防禦係數)
```
*   **物理/魔法 防禦減傷率**：計算邏輯與上述相同。在計算最終傷害時，實際造成的傷害比例為 `(1 - 防禦減傷率)`。

---

## 4. 穿透屬性機制 (Penetration)

穿透屬性（如防禦穿透、物魔穿透）的運作方式也是一個非線性曲線，用來抵消對方的防禦力。
**穿透率公式：**
```text
防禦穿透率 = 攻擊方防禦穿透值 / (攻擊方防禦穿透值 + 穿透係數)
```

**穿透對防禦的影響：**
穿透屬性並非直接相減，而是透過百分比形式削弱對方的有效防禦力或直接疊加在最終的傷害公式上。
